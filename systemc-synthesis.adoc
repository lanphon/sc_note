= SystemC可综合部分
:toc: left

== 引言

SystemC作为一种支持硬件仿真的语言，一部分子集也支持可综合设计。基于SystemC（或者其他高级语言）的可综合
设计流程，就是所谓的High-Level-Sythesis，HLS。目前各家EDA厂商对HLS的支持力度各不相同。为此，
SystemC定义了一个可综合的子集，要求支持SystemC HLS的EDA厂商至少支持这些可综合子集。
